## What kind of program is this?

I decided to write my first Python program that uses calculations. I сhose to find the number of arrangements and the number of combinations using the factorial. It doesn't solve any complex problems, but it can be useful. 

I didn't see the point in adding a graphical interface, so it only works in the terminal.
___
### What formulas does the program uses

#### Number of arrangements:
* $$(n)_k = \frac{n!}{(n-k)!} = n \cdot (n-1) \cdot \ldots \cdot (n-k+1)$$
 #### Number of combinations:
* $C_n^k = \binom{n}{k} = \frac{n!}{k!(n-k)!}$
___

:white_check_mark: If you often use factorials in your programs
